## 📖 About the Book

This repository serves as an official reference page for the book:

**MATLAB® for Brain-Computer Interface Systems: Computation and Data Processing**

The book introduces key concepts and methodologies in Brain-Computer Interface (BCI) systems, including:

- EEG signal processing  
- Feature extraction  
- Machine learning techniques  
- MATLAB-based implementation strategies
